ほんっとにはじめてのHTML_Sample15-2

リストを作ってグルーピングしよう─「番号ありリスト＜ol＞」

＜ol＞は「番号ありリスト」。
各項目に順番があるときに使います。
リスト内の各項目は＜ul＞のときと同じで＜li＞要素にします。

デフォルトだと下のように、リスト記号はアラビア数字(decimal)です。

Sample1-デフォルトの番号ありリスト

  1. 卵をよくときほぐす
  2. 卵にだし汁、薄口醤油とみりんを混ぜる
  3. 干し椎茸はもどして石づきを取り半分に切る
  4. 鶏肉は薄切りにして塩と酒をふりかける
  5. 海老は尾を残して殻をむき背わたを取る
[↓リスト部分のHTMLソース ↓]
<ol>
<li>卵をよくときほぐす</li>
<li>卵にだし汁、薄口醤油とみりんを混ぜる</li>
<li>干し椎茸はもどして石づきを取り半分に切る</li>
<li>鶏肉は薄切りにして塩と酒をふりかける</li>
<li>海老は尾を残して殻をむき背わたを取る</li>
</ol>

開始番号を指定したりできます。

Sample2-開始番号を指定する (start="数字")

  5. 海老は尾を残して殻をむき背わたを取る

Sample3-途中で番号を変更する (value="数字")

  1. 卵をよくときほぐす
  2. 卵にだし汁、薄口醤油とみりんを混ぜる
  3. 干し椎茸はもどして石づきを取り半分に切る

Sample4-数字が大きい順に表示 (reversed="数字")

  1. 1行目のテキスト
  2. 2行目のテキスト
  3. 3行目のテキスト
  4. 4行目のテキスト
  5. 5行目のテキスト

<li>3行目のテキスト</li>
<li>4行目のテキスト</li>

[↓上記のリスト部分のCSS ↓]
/* リストの装飾用CSS */
ol#sample-lower-roman li {
	list-style: lower-roman;
}
ol#sample-upper-roman {
	list-style: upper-roman;
}
ol#sample-lower-alpha {
	list-style: lower-alpha;
}
ol#sample-upper-alpha {
	list-style: upper-alpha;
}
