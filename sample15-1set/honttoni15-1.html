リストを作ってグルーピングしよう─「番号無しリスト＜ul＞」

＜ul＞は「番号無しリスト」。
一般的な箇条書きで、各項目に順番が無いときに使います。
リスト内の各項目は＜li＞要素にします。

デフォルトだと下のように、リスト記号は●(discって呼びます)です。

Sample①-デフォルトの番号無しリスト

  • オレガノ
  • カモミール
  • クレソン
  • セージ
  • タイム
  • ディル
  • バジル
  • フェンネル
  • レモングラス
  • ローズマリー
[↓リスト部分のHTMLソース ↓]
<h3>Sample①-デフォルトの番号無しリスト</h3>
<ul>
<li>オレガノ</li>
<li>カモミール</li>
<li>クレソン</li>
<li>セージ</li>
<li>タイム</li>
<li>ディル</li>
<li>バジル</li>
<li>フェンネル</li>
<li>レモングラス</li>
<li>ローズマリー</li>
</ul>

リスト記号（●などの）は、CSSで変更します。

Sample②-リスト記号-まる(circle)

  • オレガノ
  • カモミール
  • クレソン

Sample③-リスト記号-四角(square)

  • オレガノ
  • カモミール
  • クレソン

Sample④-リスト記号-なし(none)

  • オレガノ
  • カモミール
  • クレソン
[↓上記のリスト部分のCSS ↓]
/* リストの装飾用CSS */
ul#sample-circle li {
	list-style: circle;
}
ul#sample-square li {
	list-style: square;
}
ul#sample-none {
	list-style: none;
}
[↓上記のリスト部分のHTMLソース ↓]

</ul>
